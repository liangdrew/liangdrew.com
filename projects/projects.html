My Projects

Some of the projects I've had fun with.

Glide

Glide is an arcade-style mobile application for Android based on glider physics. This project was a lot of fun to make because I did not have any experience in Java, XML, or mobile development beforehand. It was quite the challenge in the beginning, but as Android Studio, object-oriented programming and Android design patterns started becoming more natural to me, I realized that I had fallen in love with Android development. You can check out Glide on the Google Play Store here and the Github repository here.

Labyrinth

Labyrinth is a maze game written in C for the Orbit Booster Pack. The mazes are uniquely solveable and randomly generated each time using an implementation of Prim's algorithm. I learned much about proper memory management and various graph algorithms while developing this project. Check out the Github repository here.

C60

C60 was the product of my sleepless work at Hack the North 2015. Built as a networking app for the Pebble watch, C60 is the optimized solution for quickly exchanging contact information with others. C60 makes use of a geolocation algorithm which identify other users within a 25 meter radius when the "Find Nearby" function is enabled. Upon finding other users, a request can be sent to exchange contact info, and upon confirmation, the names, along with other important contact details will be instantly transferred over to the respective devices. Check out the video demo here and the Github repository here.

Buzz-In

Buzz-In was one of my first side projects. Written in Python and using the Pygame framework to create a simple GUI, Buzz-In is a desktop app which allows a gamemaster to control the flow of time-based trivia games such as Reach for the Top. With a custom-made Makey Makey circuit to simulate buzzers, the software displays the team which buzzes in first, allocates points to teams that give correct answers, and prevent teams from answering the same question more than once if they give an incorrect answer the first time. Additionally, Buzz-In tracks game progress and displays team stats. Check out the Github repository here.
